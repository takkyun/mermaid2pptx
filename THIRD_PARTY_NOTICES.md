# Third-Party Notices

`mermaid2pptx` itself has **no bundled third-party source code** — it is built
with the Go standard library only, and requires no module dependencies.

This file records notices for third-party tools and material that relate to the
repository in the following, narrower senses:

- **Build/verify tooling** invoked externally (not redistributed here).
- **Generated fixtures** checked into `sample/` that embed markup produced by a
  third-party tool.

No third-party fonts are embedded: the sample SVGs reference the system font
families `"trebuchet ms"`, `verdana`, and `arial` **by name only** — no font
files are distributed in this repository.
